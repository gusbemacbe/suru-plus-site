Colophon

Technology

This site is written mostly in HTML5 and the style sheets are mostly written in CSS3. Sorry, I do not care to support such ancient browsers. WE are not in 1990s and 2000s. We are in 2020s. I encourage you to use the latest version of your favourite browser – Brave, Edge, Firefox, Google Chrome, Maxthon Browser, Opera, Safari, Samsung Browser, Vivaldi and Yandex. You must enable JavaScript to display the colours palette. The most of this site was created on VSCode Insiders 1.32.0. I do not have time to learn the interesting code editors as Emacs and Neovim that I want to try.

Typefaces

In the most of this site, I use Overpass, developed by RedHat team. Icons in the pages About and Changelog are from the font Line Awesome, developed by Icons8. The font Overpass is under GPL3, and Line Awesome is under Good Boy Licence.

CSS

I have created the own pure CSS, no frameworks. But the pure CSS mainMenu was created by Jason Knight, because I do not like to use frameworks and to load many JavaScript files, for keeping the site lightweight.

The syntax highlighting CSS file suru-plus was derived of my CSS an-old-hope, a Star Wars style, that I developed for Highlightjs.

Design

The most of icons, images and logotypes were created on Gravit Designer and now on Affinity Designer, Figma and Sketch. I love Inkscape, but the Inkscape GUI is very confusing, as in the case of layers. I am waiting for the mature development of Akira. Please, donate to Akira developers!
